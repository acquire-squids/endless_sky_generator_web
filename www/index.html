Endless Sky Generator

Choose an Endless Sky data folder

After uploading your Endless Sky data folder, click the download buttons for any plugins you may be interested in.

Full Map

A plugin that reveals every system and planet in the uploaded data folder when activated by a job available in any job board.

This works by reading all `system` root nodes. A system created by an event and defined nowhere else won't be included.
If a system is hidden or shrouded, it may not remain revealed after takeoff, if it is even revealed at all.

System Shuffler

A plugin that comes with preset randomizations of every system in the universe.

It will swap system positions but keep the shape and traversability of the universe the same.

Using the inputs below, you can configure the plugin to shuffle the universe into a random preset on certain conditions:

  • Once, immediately, upon installation
  • With a percentage chance every time you land
  • Every N days, with the shuffle happening after a greater amount of time if you have not landed for N days
Additionally, you can request a shuffle or restore at any point through any job board.

Be wary of repeated shuffling!
If you play on a version before 0.10.17's unstable release, event definitions are fully copied into your save file and your save file has potential to explode in size!
