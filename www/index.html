Full Map

A plugin that reveals every system and planet via a job available in any job board.

This works by reading all `system` root nodes.
If a system is hidden or shrouded, it may not remain revealed after takeoff.

System Shuffler

A plugin that comes with preset randomizations of every system in the universe.

It will swap system positions but keep the shape and traversability of the universe the same.
Using the inputs below, you can configure the plugin to shuffle the universe into a random preset on certain conditions:

  • Once, immediately, upon installation
  • With a percentage chance every time you land
  • Every N days, with the shuffle happening after a greater amount of time if you have not landed for N days
Additionally, you can request a shuffle or restore the universe at any point through the job board.

Be wary of repeated shuffling!
If you play on a version before v0.11.0's unstable release, event definitions are fully copied into your save file and your save file has potential to explode in size!

Don't forget to back up your saves before use!
